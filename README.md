# HDD-Activity-Agent

This add-on prevents your harddisk from stopping, so your games never get stuck at loading screens because of your hard drive.

## VERSIONS

|    VERSIONS    |            CHANGES            |            NOTES            |
|----------------|-------------------------------|-----------------------------|
